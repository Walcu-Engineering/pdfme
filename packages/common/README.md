## @walcu-engineering/pdfme-common

This library provides shared types, constants, and utility functions used across the [@walcu-engineering/pdfme-ui](https://www.npmjs.com/package/@walcu-engineering/pdfme-ui) and [@walcu-engineering/pdfme-generator](https://www.npmjs.com/package/@walcu-engineering/pdfme-generator) packages. The shared types include definitions for templates and other commonly used structures.

For the complete documentation of pdfme, please refer to [this link](https://pdfme.com/docs/getting-started).
